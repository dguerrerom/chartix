# Welcome to chartix

A toolkit for querying and managing international music charts data (1980–1999).

## Quick Start

After installation, try:

```bash
chartix list
chartix show --date 1985-06-01 --chart billboard-hot100
```

For full documentation, see the other pages.

## Features

- **CLI** – Search, anniversary hits, and more.
- **Fuzzy search** – find songs even with typos.
- **Flexible data structure** – annual CSV files with metadata.
- **Frictionless Data** – compatible with the Frictionless Data specification.
